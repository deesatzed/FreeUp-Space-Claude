## What This Project Does
This directory currently contains a flat local copy of the `mac-tools-federation` contract files, plus a newly created `GOAL.md` for the planned `mac-app-audit v0.2` build. The contract defines the shared ledger schema, safety posture, conventions, example ledger, and validator used by independent Mac tools such as `mac-app-audit` and FreeUp Space.

The intended `mac-app-audit` product is a cautious Mac software audit assistant: it should analyze app/Homebrew/software usage, write federation-valid ledger entries, produce reports, and never install, migrate, or remove software without explicit approval.

**Tech Stack:** Python 3.13.9 observed locally, JSON Schema validation via `jsonschema` when available plus a standard-library fallback in `validate_ledger.py`, Markdown contracts/docs.
**Architecture Pattern:** flat contract bundle plus planned Codex/Claude skill and command implementation. Current folder is not yet the complete skill/command implementation described by `HANDOFF_v0.1.md`.
